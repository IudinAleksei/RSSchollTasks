Информация для проверяющих

Уважаемые проверяющие, если вам не удается найти какой-либо функционал из задания в данном приложении - прошу ознакомиться с нижеприведенным описанием. Также прошу вас оставить свои контакты в отзыве.

  • Внешний вид приложения очень близок к макету, добавлены дополнительные элементы управления и слайдер
  • В прогнозе отображаются данные на неделю вперед, карточки с погодой размещены в свайпере, навигация по которому возможна стрелками(при десктопных разрешениях) и свайпом, количество карточек меняется в зависимости от ширины экрана
  • Cообщения об ошибках всех API выводятся под полем поиска
  • Элементы обновляются одновременно, есть некоторая задержка обновления background из-за анимации, если все данные, за исключением фонового изображения, не получены - страница не перерисовывается. Если не получен корректный ответ от Images API, то в качестве фонового изображения остается предыдущее(в начале - дефолтное) изображение
  • Запрос к unsplash API содержит ключевые слова соответствующие сезону и времени суток в искомом населенном пункте. Ключевые слова можно увидеть в консоли
  • Голосовое распознавание и управление выполняется на текущем языке страницы(английском или русском), за исключением белорусского (выполняется распознавание на русском), по причине некорректной работы режима continius - на белорусском результат выдается только после принудительной остановки распознавания. При смене языка страницы необходимо перезапустить голосовое распознавание
  • Громкость уведомления может быть отрегулирована командами: "quieter", "louder", "тише", "громче"
  • Голосовое уведомление о погоде можно запустить фразами: "weather", "forecast", "погода", "прогноз"
  • Дополнительный функционал: отображается УФ-индекс, реализован свайпер для отображения прогноза на 7 дней, возможно менять фон приложения речевой командой "background" или "фон"
